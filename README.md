# Elementalist-Spear-Etchings-Reffect-Pack
A more visible UI element for Elementalist's Spear Etching Mechanic

Tracks when your full charge Etching is available. 
Unfortunately, due to reffect limitations, individual charge levels are not trackable.

# How to use
First, download the zip file to your reffect root folder like so:

<img width="533" height="203" alt="image" src="https://github.com/user-attachments/assets/ed3ee674-15b9-4619-8c9a-f3113df14bb1" />

Then right click > Extract here (the necessary files will be added to the packs and icons subfolders).

Then, return to reffect ingame and refresh packs.
